💣 Campo Minado em Java

Este projeto é uma implementação do clássico jogo Campo Minado, desenvolvida em Java com execução via terminal. O objetivo principal foi praticar conceitos de programação orientada a objetos e testes automatizados com JUnit. Ele foi feito pelo curso da Cod3r e para um trabalho da faculdade na matéria de DevOps!!!

🚀 Funcionalidades
Geração de tabuleiro com minas distribuídas aleatoriamente
Abertura de campos
Marcação e desmarcação de bandeiras
Verificação de vitória e derrota
Exibição do tabuleiro no terminal
Sistema de propagação automática de áreas vazias
